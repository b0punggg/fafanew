Folder ini untuk nssm.exe (opsional, untuk install-service.bat)

Download NSSM:
  https://nssm.cc/download

Extract nssm.exe (pilih win32 atau win64 sesuai Windows) ke folder ini:
  print-bridge\tools\nssm.exe

Setelah itu jalankan install-service.bat sebagai Administrator.

Jika tidak pakai NSSM, cukup setup-kasir.bat (sudah cukup untuk kebanyakan kasus).
