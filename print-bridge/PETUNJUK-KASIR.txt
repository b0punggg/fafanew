================================================================================
  PETUNJUK SETUP PRINT BRIDGE - PC KASIR TOKOFAFA
================================================================================

Print bridge diperlukan agar nota bisa cetak ke printer thermal dari
aplikasi web (Hostinger) ke printer lokal di PC kasir.

--------------------------------------------------------------------------------
SYARAT (install sekali di PC kasir)
--------------------------------------------------------------------------------
1. Windows 7 atau lebih baru
2. Node.js LTS  -> download: https://nodejs.org
3. Printer thermal sudah terinstall di Windows
   Nama printer: BP-LITE 80D+80X Printer
4. Folder print-bridge sudah ada di PC kasir

--------------------------------------------------------------------------------
CARA SETUP (sekali saja, ~2 menit)
--------------------------------------------------------------------------------
1. Copy folder "print-bridge" ke PC kasir, contoh:
   C:\tokofafa\print-bridge

2. Double-click:  setup-kasir.bat

3. Tunggu sampai muncul "SETUP SELESAI"

4. PENTING - accept sertifikat HTTPS (sekali saja):
   Buka browser Chrome/Edge di PC kasir:
   https://localhost:3000/health
   Klik "Lanjutkan ke localhost (tidak aman)" / "Advanced" -> "Proceed"
   Ini wajib agar cetak dari https://tokofafa.dhe51.id tidak diblokir browser.

5. Selesai! Print bridge akan jalan OTOMATIS setiap:
   - PC dinyalakan
   - Kasir login Windows

--------------------------------------------------------------------------------
KASIR TIDAK PERLU:
--------------------------------------------------------------------------------
- Klik start.bat setiap hari
- Buka jendela CMD
- Install apa pun lagi (setelah setup pertama)

--------------------------------------------------------------------------------
CEK APAKAH SUDAH JALAN
--------------------------------------------------------------------------------
Double-click: cek-printer.bat
Atau buka browser: https://localhost:3000/health

Harus muncul: {"ok":true,"protocol":"https","displayName":"BP-LITE 80D+80X Printer",...}

--------------------------------------------------------------------------------
JIKA CETAK GAGAL
--------------------------------------------------------------------------------
1. Jalankan cek-printer.bat
2. Pastikan printer menyala dan kertas ada
3. Buka https://localhost:3000/health dan accept sertifikat jika diminta
4. Restart PC kasir
5. Jika masih gagal, jalankan ulang setup-kasir.bat

--------------------------------------------------------------------------------
SETUP LANJUTAN (opsional, untuk IT/developer)
--------------------------------------------------------------------------------
install-service.bat  -> jalan sebagai Windows Service (permanen)
                      perlu nssm.exe di folder tools\
                      jalankan sebagai Administrator

uninstall-kasir.bat  -> hapus auto-start Task Scheduler

--------------------------------------------------------------------------------
FILE PENTING
--------------------------------------------------------------------------------
setup-kasir.bat      -> SETUP UTAMA (jalankan ini)
cek-printer.bat      -> cek status print bridge
start-hidden.vbs     -> jalan di background (otomatis)
server.js            -> program print bridge (HTTPS)
PETUNJUK-KASIR.txt   -> file ini
